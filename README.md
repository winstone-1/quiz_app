# CLI Quiz Game

A terminal-based quiz game built with Python and Jupyter Notebook. Pick a category, answer multiple choice questions, and get scored at the end.

## How to Run

Open `main.ipynb` in VS Code and run the cell.

## Features

- 3 categories: Math, General Knowledge, History
- Questions are shuffled randomly every run
- Input validation — only accepts A, B, C or D
- Shows the correct answer when you get one wrong
- Missed questions summary at the end
- Score feedback based on your result

## Scoring

| Score | Feedback |
|-------|----------|
| 100% | Perfect Score! |
| 80% and above | Great job |
| 60% and above | Fair job |
| 40% and above | Do better next time |
| Below 40% | Keep studying! |

## Requirements

- Python 3.x
- Jupyter Notebook or VS Code with Jupyter extension
